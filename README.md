## Overview
**Bot BCC1** is an automated script built natively on **Puppeteer**. It is designed to navigate directly to Betika, log in automatically using secure credentials from `.env`, and proceed straight to the Crash Games section—specifically launching the "Comet Crush" game. 

This bot has been customized to simulate realistic, human-like activity including variable typing speeds, mouse focus clicks, and natural delays to seamlessly operate while evading basic bot protections.

## Features
- **Stealth Browsing:** Utilizes `puppeteer-extra` and `puppeteer-extra-plugin-stealth` to appear fully human.
- **Human Actions:** Delays, realistic typing simulations, and event-focused workflow interactions.
- **Secure Credentials Management:** Operates using environment variables (`.env`) for both credentials and network configurations.
- **Proxy Support:** Supports custom proxy assignments (`PROXY_URL`) for an added layer of geographic/request security.
- **Automated Navigation:** Zero-click necessary workflow. Starts from Login into Crash Games and Comet Crush.

Note: The use of automated scripts on gambling and game sites often contravenes their explicit Terms of Service. This script is intended purely as a programming exercise.
